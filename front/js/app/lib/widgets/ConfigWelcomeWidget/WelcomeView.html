Congratulations, you have successfully installed and launched the DataTorrent Gateway. This Gateway is the interface to the DataTorrent Platform.

In order to complete installation of DataTorrent, the following are required:

  • YARN and DFS configured and running

If the above prerequisites have been completed, we will take you through a few configuration steps on the following screens. These steps include:

  • DataTorrent License Confirmation
  • Hadoop and DFS Configuration
